# US Stock Algorithmic Trading

US Stock Algorithmic Trading is a program for projecting buy and sell signals for US Stocks (users' choice) from January 2016 until present using SMA50 and SMA200 crossover strategy.

## Installation
Use the package manager [pip](https://pip.pypa.io/en/stable/installation/) to install pandas, numpy, matplotlib, and datetime.
